SOFTWARE

Software: Conjunto de instrucciones y datos
que indican al hardware cómo realizar tareas específicas.
Se divide en dos categorías principales:
software de sistema (como sistemas operativos)
y software de aplicación (programas que realizan tareas específicas).

Código Fuente:

Instrucciones escritas en un lenguaje de programación que define al software.

Documentación:

Manuales, guías y comentarios en el código para entender y utilizar el software.

Interfaz de Usuario (UI):

Es la parte del software con la que interactúan los usuarios, como ventanas, botones y menús.

Base de Datos:

Almacena y gestiona los datos que el software necesita para funcionar.

Algoritmos:

Son las secuencias de pasos lógicos que el software sigue para resolver problemas específicos.

Bibliotecas:

Conjuntos de funciones y herramientas reutilizables que facilitan el desarrollo de software.

Entorno de Desarrollo:

Este es el contenido que aparece al hacer clic en la imagen.

Frameworks:

Estructuras de soporte que proporcionan una base para desarrollar aplicaciones.

Controladores de Dispositivos:

Programas que permiten la comunicación entre el sistema operativo y el hardware.
