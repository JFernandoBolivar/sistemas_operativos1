FIRMWARE

Firmware: Es un software esencial para el funcionamiento básico de un dispositivo, ya que está incrustado directamente en su hardware. A diferencia del software regular, el firmware se almacena en memoria no volátil, lo que asegura que no se pierde cuando el dispositivo se apaga.

ROM (Memoria de Sólo Lectura):

Memoria es no volátil donde se almacena el firmware.

Cargador de Arranque (Bootloader):

Su función principal es cargar el sistema operativo o el firmware principal.

Código del Firmware:

Conjunto de instrucciones que controla el hardware del dispositivo. Con funciones para que el dispositivo opere correctamente.

Capa de Abstracción de Hardware (HAL):

Es puente entre el hardware y el software, facilitando que el firmware maneje el hardware de manera óptima.

Actualizaciones de Firmware:

Permiten corregir errores, mejorar el rendimiento o añadir nuevas funciones al dispositivo.

Interfaz de Usuario (UI):

En ciertos dispositivos, el firmware ofrece una interfaz para que los usuarios interactúen y configuren el dispositivo.
