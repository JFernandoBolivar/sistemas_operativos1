Caracteristicas

Caracteristicas: Las características de un sistema operativo son atributos o capacidades que permiten que el sistema cumpla con sus funciones de manera efectiva.

Multitarea:

Permite ejecutar múltiples tareas simultáneamente.

Gestión de memoria:

Optimiza el uso de la memoria del sistema.

Interfaz de usuario:

Ofrece una forma interactiva para que los usuarios interactúen con el sistema.

Seguridad:

Protege los datos y recursos del sistema contra accesos no autorizados.

Sistema de archivos:

Protege los datos y recursos del sistema contra accesos no autorizados.

Soporte de red:

Facilita la conexión y comunicación entre múltiples dispositivos en una red.
